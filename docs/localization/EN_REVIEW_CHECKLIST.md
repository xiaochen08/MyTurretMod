## 1. Language Naturalness
- [ ] Wording sounds native to US players.

## 2. Style Standards
- [ ] Uses American spellings (`color`, `center`, `armor`, etc.).
- [ ] Avoids culturally confusing references.
- [ ] Humor/slang is understandable to US players and not overused.

## 3. Game Terminology
- [ ] Combat, upgrade, and command terms are clear.
- [ ] Item/module names are consistent in all screens.
- [ ] Command help text is accurate and concise.

## 4. UX Safety
- [ ] No text truncation or overlap on common resolutions.
- [ ] Placeholder tokens remain correct (`%s`, `%d`).
- [ ] Language switch + `F3+T` refreshes text without stale strings.
